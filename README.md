## 🤞 Iniciando o projeto

**Clone o projeto e acesse a pasta**

```bash
git clone https://github.com/vinive/ignite-lab-react.git && cd ignite-lab-react
```

**Siga os passos**

```bash
# Instale as dependências
$ npm install

# Inicie a aplicação
$ npm run dev
```
